# Detection: Suspicious Encoded PowerShell Command

## Why I Built This Detection
Attackers commonly use the PowerShell -EncodedCommand flag to 
Base64 encode malicious scripts and bypass basic string-based 
detections. I built this detection to catch any use of encoded 
commands on the endpoint regardless of what the decoded content 
contains.

## Search Query
index="windows" source="wineventlog:microsoft-windows-powershell/operational"
EventCode=4104 Message="EncodedCommand"

## Why I Used Event ID 4104
Event ID 4104 is generated by PowerShell Script Block Logging and 
captures the decoded content of any PowerShell command including 
encoded ones. This means even if an attacker encodes their payload 
Splunk will log what the command actually does after decoding.

## Alert Settings
- Schedule: Real-time
- Trigger: Number of results > 0
- Severity: High
- Throttle: 60 minutes
- Action: Add to Triggered Alerts
- Expiration: 24 hours
